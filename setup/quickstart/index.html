Quick Start Guide

This Quick Start guide provides a comprehensive tutorial designed to step you through WebLogic Kubernetes Toolkit UI (WKT UI) functionality and familiarize you with its capabilities. WKT UI is a desktop application that helps you create and deploy WebLogic domains into Kubernetes. Using this guide, you will deploy a sample WebLogic-based application to a Kubernetes cluster.

Note that this walk-through is for demonstration purposes only, not for use in production. These instructions assume that you are already familiar with Kubernetes.
